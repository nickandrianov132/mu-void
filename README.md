# MU-VOID 🌌 
**Fullstack-платформа для MMORPG MU Online**

Профессиональный веб-портал, интегрированный с игровой экосистемой MU Online. Проект решает задачи автоматизации игровых процессов: от регистрации и рейтингов до сложных систем лояльности и приема платежей.

📍 **Live Demo:** [https://www.mu-void.com](https://www.mu-void.com)

---

## 🔥 Ключевые технические решения

### 1. Синхронизация серверного времени (Time Sync)
Реализован алгоритм компенсации разницы во времени между сервером (UTC-0) и клиентом.
*   **Проблема:** Рассинхрон локальных часов пользователя мешает точному отображению игровых ивентов.
*   **Решение:** При загрузке вычисляется `offset` (смещение) относительно серверного времени. Использование `setInterval` с очисткой (`cleanup function`) в React-хуках позволяет отображать точное время без утечек памяти.

### 2. Безопасность и работа с данными (MSSQL)
Взаимодействие с базой данных реализовано через пакет `mssql` с фокусом на безопасность:
*   **Защита от SQL-инъекций:** Использование параметризованных запросов (`.input()`) и вызов хранимых процедур.
*   **Бизнес-логика:** Реализована система покупки и конвертации VIP-статуса с динамическим пересчетом остаточной стоимости валюты.

### 3. Авторизация и State Management (RTK Query)
Централизованное управление состоянием и сетевыми запросами через **Redux Toolkit**:
*   **JWT Auth:** Автоматическая подстановка токена в заголовки через `prepareHeaders`.
*   **Reauth Interceptor:** Кастомный `baseQueryWithReauth` для обработки 401 ошибок и бесшовного логаута/обновления сессии.

## 🚀 Функционал
- **User Panel:** Управление персонажами, сброс характеристик (Reset/Grand Reset) с системой условий.
- **Rankings:** ТОП-100 игроков, владельцы замков и онлайн-статус, характеристики персонажа в реальном времени.
- **Monetization:** Интеграция платежных шлюзов **PayPal** и **CryptoCloud**.
- **Vote System:** Автоматическое начисление вознаграждения (WCoins) через систему Postback от игровых рейтингов.
- **Community:** Интеграция виджета Discord и динамические таймеры событий.

## 🛠 Технологический стек

**Frontend:**
- React 18, React Router 6
- State: Redux Toolkit, RTK Query
- Styles: SCSS (модульная архитектура)
- Инструменты: Vite

**Backend:**
- Node.js, Express
- DB: MSSQL (Microsoft SQL Server)
- Auth: JWT (JSON Web Tokens)

## 📦 Установка и запуск

1. **Клонирование:**
   ```bash
   git clone https://github.com/nickandrianov132/mu-void.git

2. **Настройка клиента:**
    cd client
    npm install
    # Настройте VITE_API_URL в .env 
    # VITE_API_URL = 'https://www.your-domain.com/' or 'http://localhost:7000/'
    npm run dev
3. **Настройка Server:**
    cd server
    npm install
    # Укажите параметры подключения к MSSQL в .env (DB_NAME, DB_HOST, DB_USER, DB_PASSWORD,DB_PORT)
    npm run dev
